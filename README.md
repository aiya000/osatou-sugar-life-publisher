# :two_hearts: お砂糖届け発行所 :two_hearts:
## Development
### Requirements

- node v16.15.0 (recommended: `nvm install v16.15.0`)

### Run doctests

```shell-session
$ npm run doctest
```

NOTE: This project is using tsdoc-testify to run doctests.

### Take screenshots

Currently, using this is not for tests.

```shell-session
$ npm run screenshot
```
